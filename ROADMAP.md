# B.L.I.N.K. Roadmap

Current version: **v2.8.8** (Feb 2026)

---

## Completed

- [x] Core LED wall calculator (panels, processors, power, data, structure)
- [x] Multi-screen project support
- [x] Canvas visualization with drag-and-drop positioning
- [x] 5 layout views (standard, power, data, structure, gear list)
- [x] Export: PDF, PNG/JPEG, email gear list, Resolume XML, .ledconfig
- [x] PWA — offline-capable, installable, auto-update notifications
- [x] User authentication (Supabase — sign up, sign in, password reset)
- [x] Custom panels & processors (local + cloud sync)
- [x] Community library (submit, browse, download, admin approval)
- [x] Simple & Complex mode with welcome page
- [x] Touch gestures & keyboard shortcuts
- [x] Undo/redo system
- [x] Unit conversion (Imperial ↔ Metric)
- [x] B.L.I.N.K. branding & custom domain
